## タスク

以下のニュース記事リストを分析し、このユーザーにとって読む価値がある記事を選別してください。

各記事について、以下の観点で10点満点でスコアをつけてください:

- **関連性** (0-4点): このユーザーの業種・役職に直接関係するか
- **緊急性** (0-3点): 今日・今週の経営判断に影響しうるか
- **実用性** (0-3点): 具体的なアクションや意思決定のヒントになるか

## 出力形式

必ず以下のJSON形式で回答してください。JSON以外のテキストは出力しないでください。

```json
[
  {
    "index": 0,
    "score": 8,
    "reason": "原材料コスト上昇は製造業の経営者として直ちに対応が必要な情報",
    "one_line_summary": "鉄鋼・アルミ価格が5%上昇、製造業のコスト圧迫が続く見通し"
  }
]
```

- `index`: 入力記事リストのインデックス番号（0始まり）
- `score`: 合計スコア（0〜10）
- `reason`: このユーザーにとってなぜ重要か（50字以内）
- `one_line_summary`: ユーザー視点での一言要約（60字以内、記事タイトルをそのまま使わず意訳する）

スコアが{{min_score}}点未満の記事は含めないでください。
上位{{max_articles}}件のみ返してください（スコアが高い順）。
